Keystone Gateway

Der intelligente Reverse Proxy für moderne DevOps-Teams

Gateway läuft erfolgreich

Blitzschnell

300+ req/sec mit <5ms Latenz

Einfach

YAML-Konfiguration, Single Binary

Erweiterbar

Optional: Lua-Scripts für CI/CD

Quick Start

1. Konfiguration erstellen:

# config.yaml
tenants:
- name: "my-app"
domains: ["app.example.com"]
services:
- name: "backend"
url: "http://localhost:3000"

2. Gateway starten:

keystone-gateway -config config.yaml -addr :8080
