From zero to routing in 60 seconds

1. Configure tenants

config.yaml
tenants: - name: "api" domains: - "api.company.com" lua_routes: "api" services: - name: "backend" url: "http://api-server:3000"

2. Write Lua routes

scripts/api.lua
-- Route with auth middleware chi_middleware(function(req, res, next) if request_header(req, "Authorization") then next() else response_status(res, 401) end end) chi_route("GET", "/users", function(req, res) response_write(res, "User data") end)
# Start the gateway ./keystone-gateway -config config.yaml
