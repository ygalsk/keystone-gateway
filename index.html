Quick Start

Configuration

# config.yaml
tenants:
  - name: "api"
    domains: ["localhost"]
    lua_routes: "api"
    services:
      - name: "backend"
        url: "http://localhost:3001"

Lua Script

-- scripts/api.lua
chi_route("GET", "/hello", function(request, response)
    response:write("Hello World")
end)

Start: ./keystone-gateway -config config.yaml

Philosophy

Keep it simple. Get it working. Make it fast.
