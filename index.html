From zero to production-ready reverse proxy in minutes. No complexity, just results.

Docker Deployment

1. Pull the image

docker pull ygalsk/keystone-gateway:latest

2. Create config.yaml

routes:
  - path_prefix: "/api/"
    backend: "http://localhost:3000"

3. Run the container

docker run -d -p 8080:8080 \
  -v $(pwd)/config.yaml:/app/config.yaml \
  ygalsk/keystone-gateway:latest

Health checks, graceful shutdown, and monitoring included out of the box.

Development Setup

1. Clone repository

git clone https://github.com/ygalsk/keystone-gateway.git
cd keystone-gateway

2. Build and test

make build
make test

3. Run locally

./keystone-gateway -config=configs/config.yaml
